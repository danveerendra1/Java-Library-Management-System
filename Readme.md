# 📚 Library Management System

A console-based **Library Management System** developed in Java as part of the **Elevate Labs Java Developer Internship**.

This project demonstrates the practical implementation of **Object-Oriented Programming (OOP)** concepts including Encapsulation, Abstraction, and Polymorphism through a menu-driven application for managing books and users.

---

## 🚀 Features

### 📖 Book Management
- Add New Book
- View All Books
- Search Book by ID
- Prevent Duplicate Book IDs

### 👤 User Management
- Add New User
- View All Users
- Prevent Duplicate User IDs

### 📕 Library Operations
- Issue Book
- Return Book
- Book Availability Status

## ▶️ How to Run

Clone the repository

```bash
git clone https://github.com/danveerendra1/Java-Library-Management-System
```

Navigate to src folder

```bash
javac *.java
```

Run the application

```bash
java Main
```
